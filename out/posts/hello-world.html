欢迎来到我的个人博客！这是博客的第一篇文章。

为什么开始写博客？

写博客是我一直以来的想法，主要有几个原因：

  1. 记录学习：作为开发者，记录学习过程中遇到的问题和解决方案
  2. 分享知识：将自己掌握的技术分享给更多人
  3. 提升表达：写作是锻炼思维和表达能力的好方式

博客技术栈

这个博客使用以下技术构建：

  • 前端：Next.js + TypeScript + Tailwind CSS
  • 后端：Java Spring Boot
  • 数据库：MongoDB
  • 内容：Markdown 文件管理
